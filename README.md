# Meme Dealers

## Getting Started
> - Step 1: Fork the Repository
> - Step 2: Have the following modules installed:
> - - random
>   - os
>   - streamlit
> -  Step 3: run the command "streamlit run "main.py" in your python command prompt
> - Step 4: Press button and Enjoy Meme :D
